### Important setup commands
```git config --global user.name "[firstname lastname]"``` <br>
```git config --global user.email "[email]"```

### Initialize repo
```git init```

### Clone repo
```git clone [url]```

### Stage a file
```git add [file]```

### Unstage a file
```git reset [file]```

### Show changes
```git diff``` <br>
```git diff --staged```

### Create a branch
```git branch [name]```

### Merge branches
```git merge [branch]```

### List existing branches
```git branch```

### Inspect commit history
```git log```

### Upload to repo
```git push [repo] [branch]``` <br>
```git push [repo] --all```

### Fetch from repo
```git pull```

### Remove file and stage its removal
```git rm [file]```

### Move file path and stage its move
```git mv [path] [new path]```
