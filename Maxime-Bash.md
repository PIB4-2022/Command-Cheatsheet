### Commit changes
```git commit -m "[message]"```

### Commit changes to a branch
```git commit -m "[message]" [branch]```

### Create a tag
```git tag [name]```

### Push a tag
```git push [repo] [tag]```

### Push all tags
```git push [repo] --tags```

### Remove a tag
```git tag -d [tag]```

### Remove a tag from remote
```git push [repo] :refs/tags/[tag]```

### Checkout a branch
```git checkout [branch]```

### Checkout a tag
```git checkout [tag]```

### Checkout a commit
```git checkout [commit]```
